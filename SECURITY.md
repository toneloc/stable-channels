# Security Policy

## Supported Versions

Security updates are provided for the latest publicly available version of Stable Channels. Please update to the newest release before reporting an issue.

## Reporting a Vulnerability

To report a potential security vulnerability, email:

**[support@stablechannels.com](mailto:support@stablechannels.com)**

Please include:

* A description of the vulnerability
* Steps to reproduce it
* The affected platform and app version
* Any relevant logs, screenshots, or supporting information

Please do not publicly disclose the vulnerability until we have investigated and addressed it. We will acknowledge your report and provide updates as our investigation progresses.
